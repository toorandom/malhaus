What is malhaus?

malhaus is a self-hosted malware static triage platform. Submit a suspicious file or URL and it tells you whether it is malware, what it does, and why it thinks so — without executing the file.

It combines three things that are usually separate: classic static analysis CLI tools run automatically per file type; deterministic heuristics that score evidence without any LLM; and an LLM verdict loop that receives the full evidence pack, can request additional tool calls, and returns a final verdict with confidence and top reasons.

How it works

1 · Preflight
File type is detected by magic bytes first, extension second. Type-specific CLI tools run automatically — authenticode, objdump, radare2 for PE; readelf and ldd for ELF; oledump and olevba for Office; and so on. Archives are extracted and the inner payload promoted to full analysis.
2 · Heuristics
Entropy, IOC patterns, encoded string detection, embedded payload extraction, and import name matching produce a numeric risk score (0–100) and a risk hint — all deterministic, no LLM involved.
3 · LLM verdict
The LLM receives the evidence pack and can call additional tools (up to 6 times) before returning a final verdict: risk_level, confidence, top_reasons, and extracted IOCs. A faster model pre-screens raw strings before the verdict pass.
4 · Visualizations
Every report includes an entropy profile, compression curve, byte bigram matrix, and an interactive 3D byte trigram point cloud (WebGL) with a PCA 2D projection — all computed from raw bytes, no execution required.

Supported file types

pe Windows executables and DLLs — authenticode, objdump, radare2, FLOSS, optional Ghidra
elf Linux binaries — readelf, objdump, ldd, optional Ghidra
office doc / xls / rtf — oledump, olevba, oleobj, rtfobj
office_openxml docx / xlsx / pptx — macro and relationship extraction
pdf PDF — keyword scan, pdfid, pdf-parser
msi Windows installers — 7z extraction, inner PE analysis
lnk Windows shortcuts — target, arguments, metadata
archive zip / 7z / rar — extraction, inner payload promotion, password support
ps1 / vbs / js / sh Scripts — content analysis, base64 blob extraction

Who made this

malhaus was built by Eduardo Duarte, a mathematics researcher and security practitioner who did his PhD at the University of Groningen.

The name is a play on Bauhaus — triage as craft. The byte trigram visualization is an original application of delay-coordinate geometry (inspired by Takens' theorem) to malware byte streams.

Access

Web
CAPTCHA-protected, per-IP rate limited. Results cached by SHA-256 — submitting the same file twice returns instantly.
REST API
Bearer token authentication, async job polling, per-key rate limits. API results appear in the web index alongside browser submissions. See README-API.md.
MCP
Remote MCP server with OAuth 2.0 — AI agents (Claude, Cursor, Copilot, Semantic Kernel) can call the analyze tool natively. See README-MCP.md.
